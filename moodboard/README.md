# moodboard

Drop reference screenshots (PNG/JPG) here — Inshorts-style cards, dark
editorial apps, type-driven layouts, anything whose look you want dig.ai's
feed to have.

When the Phase 2 design pass runs, images in this folder are treated as the
**primary design reference** and `design/MOODBOARD.md` is derived from them.
If the folder is empty, the direction comes from web research instead.
